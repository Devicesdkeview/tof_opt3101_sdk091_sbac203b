Detailed Description

Class that enumerates all registers in OPT3101.

Class that enlists and declares all OPT3101 registers accesible to user. The names of the registers match with names in the datasheet These registers are directly accesible to the user in the program.
For example: OPT3101Registers::hdr_mode = 1 would set the register hdr_mode to 1
Similarly to read the value of the register users need to call the devRegister::read() function like OPT3101Registers::hdr_mode::read()
